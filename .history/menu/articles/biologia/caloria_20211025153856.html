Caloria

writer: Daniel Matos

Uma caloria é a quantidade de calor necessário para elevar a temperatura de uma grama de água de 15,5 para 15,5 graus celcios. 1 cal = 4,1855 jaules, jaule é a unidade que mede calor. Geralmente quando nos referimos a caloria estamos nos referindo a 1000 calorias, que é a kilo calorias(kcal).

Isso significa que a principal função dos alimentos que comemos é queima-los para assim gerar calor, calor este que é utilizado por todo nosso corpo para manter tudo funcionando. Todo o metabolismo depende de calor. Quando comemos além do necessário então armazenamos esse alimento, para quando tivermos de menos podermos queima-los e gerar calor. Para o metabolismo funcionar precisamos de calor.
